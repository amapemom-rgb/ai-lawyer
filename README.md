# AI Lawyer — ИИ-Юрист

Интеллектуальная юридическая система с понятийно-графовой архитектурой.

## Три духа проекта

- **Безопасность** — данные зашифрованы, доступ контролируется, ошибки ловятся на каждом уровне
- **Вирусность** — адаптивность, лёгкость внедрения, бесперебойность, незаменимость, всепокрываемость, распространяемость
- **Интеллектуальность** — система-«почемучка», которая уточняет, учится и наращивает граф знаний

## Архитектура

```
Пользователи
    │
    ▼
ИИ-Агент (сменяемый, чистый контекст)
    │
    ▼
Понятийно-Графовая Система (ПГС)
   ╱          ╲
  ▼            ▼
База знаний   База данных
(ChromaDB)    (PostgreSQL)
    ▲
    │
Входной конвейер (парсинг → классификация → загрузка)
```

## Стек технологий

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| LLM | Claude API (Anthropic) |
| Граф (ПГС) | Neo4j |
| База знаний | ChromaDB (векторное хранилище) |
| База данных | PostgreSQL |
| Фреймворк агента | LangChain / Anthropic SDK |
| API | FastAPI |
| Контейнеризация | Docker Compose |

## Структура проекта

```
ai-lawyer/
├── src/
│   ├── agent/          # ИИ-агент (LLM-ядро, почемучка, валидатор)
│   ├── pgs/            # Понятийно-графовая система
│   ├── knowledge/      # База знаний и векторное хранилище
│   ├── pipeline/       # Входной конвейер документов
│   ├── database/       # Модели и миграции PostgreSQL
│   └── api/            # FastAPI эндпоинты
├── docs/               # Документация
├── tests/              # Тесты
├── docker-compose.yml
├── requirements.txt
└── .env.example
```

## Быстрый старт

```bash
git clone https://github.com/amapemom-rgb/ai-lawyer.git
cd ai-lawyer

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

cp .env.example .env
# Заполнить .env своими ключами

docker-compose up -d

python -m src.api.main
```
