# Быстрый старт — Фаза 1

## Шаг 1: Получить API-ключи

Тебе нужно два ключа:

### 1.1 Anthropic API Key (для Claude)

1. Перейди на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйся (или войди)
3. Перейди в **Settings → API Keys**
4. Нажми **Create Key**
5. Скопируй ключ (начинается с `sk-ant-...`)
6. Пополни баланс в **Billing** (для старта достаточно $5–10)

Стоимость: Claude Sonnet — примерно $3 на миллион входных токенов. Для MVP это очень дёшево.

### 1.2 Telegram Bot Token

1. Открой Telegram и найди бота [@BotFather](https://t.me/BotFather)
2. Отправь ему `/newbot`
3. Введи имя бота (например: `ИИ Юрист`)
4. Введи username бота (например: `ai_lawyer_test_bot`) — должен заканчиваться на `bot`
5. BotFather даст токен вида `7123456789:AAH...`

Это бесплатно.

## Шаг 2: Настроить проект

```bash
# Клонировать репозиторий
git clone https://github.com/amapemom-rgb/ai-lawyer.git
cd ai-lawyer

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или: venv\Scripts\activate  # Windows

# Установить зависимости
pip install -r requirements.txt

# Создать файл настроек
cp .env.example .env
```

Открой `.env` и заполни свои ключи:
```
ANTHROPIC_API_KEY=sk-ant-...твой-ключ...
TELEGRAM_BOT_TOKEN=7123456789:AAH...твой-токен...
```

## Шаг 3: Загрузить начальную базу знаний

```bash
python scripts/load_initial_docs.py
```

Это загрузит в ChromaDB (локально, без Docker) стартовый набор документов:
закон о защите прав потребителей, ГК РФ (ключевые статьи),
правила маркетплейсов и др.

## Шаг 4: Запустить бота

```bash
python -m src.bot.telegram
```

Готово! Открой своего бота в Telegram и напиши ему вопрос.

## Шаг 5: Проверить

Попробуй задать боту эти вопросы:

1. «Маркетплейс потерял мой товар. Что делать?»
2. «Как написать претензию о возврате денег за брак?»
3. «Какие сроки возврата товара по закону РФ?»

Если бот отвечает со ссылками на конкретные статьи — Фаза 1 работает.
